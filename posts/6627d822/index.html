JetBrains旗下IDE最新UI抢先体验

发表于 2022-06-02 分类于 Tool 评论数： 阅读次数： 本文字数： 375 阅读时长 ≈ 1 分钟

前言

IntelliJ IDEA 觉得过去的20年UI变化太小，所以决心改版，搞一套现代化UI。预计明年为新用户默认，老用户逐步推送。旧版UI大概保留一年，从此旗下产品都采用新UI。

原文连接：https://blog.jetbrains.com/zh-hans/idea/2022/05/take-part-in-the-new-ui-preview-for-your-jetbrains-ide/

新UI目前还处在邀请体验阶段，特共享无邀请启用方式：

效果图

体验方式

  1. 下载最新UI插件：https://wwt.lanzouf.com/in1bJ05rvjng

  2. 保证 JetBrains IDE 的版本为 2022.2 EAP 否者会提示你版本不匹配

    EAP 下载地址：https://www.jetbrains.com/resources/eap/

  3. 安装插件到 IDE

  4. 重启 IDE 即可
