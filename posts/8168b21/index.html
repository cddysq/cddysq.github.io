搭建UnblockNeteaseMusic解锁网易云音乐客户端变灰歌曲

发表于 2022-05-20 分类于 Tool 评论数： 阅读次数： 本文字数： 656 阅读时长 ≈ 2 分钟

简述

解除网易云灰色音乐(无版权)歌曲的工具，让你可以听取客户端任何的歌曲，而不受到限制。

截止目前，出现小部分歌曲仅能试听。

项目地址：Unlock-netease-cloud-music

安装

由于项目作者对于windows下的搭建方式已经清楚说明，本文只演示Linux下的搭建过程。

源码准备

下载源码(任选其一)

方式一：下载到本地后上传到服务器

方式二：服务器直接下载

#最后的指定目录可根据需求自行替换
git clone https://github.com/meng-chuan/Unlock-netease-cloud-music.git /usr/local/

修改配置

打开Unlock-netease-cloud-music\src\app.js修改端口号。

环境安装

安装Node.js

#Debian/Ubuntu系统
curl -sL https://deb.nodesource.com/setup_10.x | bash -
apt install -y nodejs git 

#CentOS系统
curl -sL https://rpm.nodesource.com/setup_10.x | bash -
yum install nodejs git -y

安装pm2

npm install pm2 -g
ln -s /usr/local/nodejs/bin/pm2 /usr/local/bin/

验证安装

启动应用

在服务器的UnblockNeteaseMusic源码目录下执行命令：

pm2 start app.js
